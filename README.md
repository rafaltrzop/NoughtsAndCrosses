Noughts and Crosses
===================

Noughts and Crosses also known as [Tic-tac-toe](http://en.wikipedia.org/wiki/Tic-tac-toe) is a simple game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three respective marks in a horizontal, vertical, or diagonal row wins the game.

Game features
=============

* Single player mode (you vs. computer) with 3 difficulty levels
* Multiplayer mode (play against your friend)
* Stunning console like graphics

How to compile it?
==================

Just use attached Makefile and type in console `make all` or in short `make`.
To get rid of object files and executable file type `make clean`.
